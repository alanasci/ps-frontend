# Título do Projeto

Este projeto é o frontend de uma aplicação de feedback entre pais e professores com relação a seus alunos.


## Instalação

Instale o projeto com npm

```javascript
  npm install
```

Depois inicie o projeto com o comando
```javascript
  npm run dev
```
